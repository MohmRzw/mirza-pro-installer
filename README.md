## شروع سریع نصب آسان میرزا پرو:

1. اسکریپت را دانلود کنید:

```bash
git clone https://github.com/mohmrzw/mirza-pro-installer.git
cd mirza-pro-installer
```

2. اجازه اجرا بدهید:

```bash
chmod +x installer.sh
```

3. اجرا کنید و دستور نصب آسان را دنبال کنید:

```bash
sudo ./installer.sh
```

منوی تعاملی باز می‌شود و گزینه‌ها شامل:

* نصب کامل Mirza Pro
* حذف کامل ⚠️ (در دست توسعه)
* آپدیت + فیکس خودکار ⚠️ (در دست توسعه)
* ویرایش config.php
* وضعیت وب‌هوک
* خروج

## نکات مهم
* رمز دیتابیس در مسیر `/root/mirza_pass.txt` ذخیره می‌شود، حتماً آن را امن نگه دارید.
